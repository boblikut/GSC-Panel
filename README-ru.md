# GSC - Панель

### Проект разработан [Vantuzz](https://steamcommunity.com/id/vantuzz_nya/)

## Вступление
Это панель администратора для серверов Garry's Mod. Ваши администраторы смогут выполнять свои обязанности даже без возможности зайти на сервер Garry's Mod, через удобный веб-интерфейс.

## О проекте
Проект представляет собой самостоятельное веб-приложение + скрипты для Garry's Mod. Это означает, что вам придётся размещать этот проект на собственном хостинге или иным способом.

## Использование
Прежде всего, вам нужно как-то войти в учётную запись администратора. Это просто сделать. По умолчанию в базе данных уже создана следующая учётная запись:

Логин: `owner`
Пароль: `1234`

Сразу после входа в учётную запись смените пароль. Для этого перейдите в Панель администратора, пользователи, и нажмите на карандаш рядом с логином "owner". Там же вы можете изменить логин.

Чтобы создавать учётные записи для ваших администраторов, нажмите «Создать учётную запись» на вкладке пользователей.

Чтобы редактировать роли и создавать новые, перейдите на вкладку «Роли» и сделайте это. Также вы можете создавать собственные команды для игроков. Проверьте `gsc/plugins/commands_example.lua` для примеров. Не забудьте синхронизировать права ролей с командами игроков в Панели администратора -> Пользователи -> Синхронизировать права.

Также вы можете настроить возможность для администраторов просматривать пользовательскую информацию об игроках. Для этого ознакомьтесь с `gsc/plugins/info_example.lua` с примерами.

## Установка
0. Скачайте `Docker`
1. Запустите `Docker`
2. Загрузите образ Docker, введя команду: `docker pull ghcr.io/boblikut/gsc-panel:latest`
3. Перенесите папку `gsc` из `addons` в папку `addons` на вашем сервере gmod
4. Откройте `gsc/sv_gsc-panel.lua` и измените `ws://localhost:8080` на актуальный адрес веб-сокета (`ws://[ДОМЕН or IP]:[ПОРТ](по умолчанию 8080)`)
5. Скачайте [GWSockets](https://github.com/FredyH/GWSockets) и поместите .dll из нужного вам релиза в `lua/bin`
6. Сгенерируйте токен в Админке -> Токен и поместите его в `data/gsc/gsc-token.txt` вашего сервера gmod.

## Запуск
1. Создайте и запустите контейнер docker с помощью команды: `docker run --name gsc-panel -d --restart always -p 8080:8080 -p 80:80 -v gsc-db-volume:/var/www/html/db ghcr.io/boblikut/gsc-panel:latest`
2. Чтобы перезапустить контейнер, введите: `docker restart gsc-panel`
3. Чтобы остановить созданный контейнер: `docker stop gsc-panel`
4. Чтобы запустить созданный контейнер: `docker start gsc-panel`

*В Linux напишите `sudo` перед `docker`*

Если вы правильно установили дополнение и сервер запущен, вы сразу увидите актуальную статистику, игроков и т.д.
